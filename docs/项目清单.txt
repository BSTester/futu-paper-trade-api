🔧 核心代码文件

✅ main.py                  FastAPI主程序，定义所有API接口
   - 系统接口（健康检查、根路径）
   - 账户接口（账户列表、账户详情）
   - 持仓接口（持仓查询）
   - 行情接口（搜索股票、获取行情、热门股票）
   - 交易接口（下单、买入、卖出）

✅ futu_client.py           富途API客户端，封装业务逻辑
   - 账户管理（获取账户列表、账户详情）
   - 持仓查询（获取持仓列表）
   - 行情查询（搜索股票、获取行情、热门股票）
   - 交易操作（下单交易）

✅ models.py                Pydantic数据模型定义
   - AccountInfo: 账户信息模型
   - Position: 持仓信息模型
   - StockQuote: 股票行情模型
   - TradeRequest: 交易请求模型
   - TradeResponse: 交易响应模型
   - StockSearchResult: 股票搜索结果模型

✅ config.py                配置文件
   - 环境变量加载
   - API端点配置
   - 市场类型映射（美股/港股/A股）
   - 订单类型映射（限价单/市价单）
   - 证券类型映射（股票/ETF/期权）

🎯 API接口清单
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

系统接口:
   GET  /                          根路径
   GET  /health                    健康检查

账户接口:
   GET  /api/accounts              获取账户列表
   GET  /api/account/info          获取账户详情

持仓接口:
   GET  /api/positions             获取持仓列表

行情接口:
   POST /api/stock/search          搜索股票
   GET  /api/stock/quote           获取股票行情
   GET  /api/stock/hot             获取热门股票

交易接口:
   POST /api/trade                 下单交易（完整接口）
   POST /api/trade/buy             快速买入
   POST /api/trade/sell            快速卖出

📖 快速开始步骤
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1️⃣  安装依赖
   双击运行: setup.bat
   或执行: pip install -r requirements.txt

2️⃣  获取Cookie
   - 访问 https://www.futunn.com/paper-trade
   - 登录账户
   - F12 → Network → 复制Cookie

3️⃣  配置环境
   - 复制 .env.example 为 .env
   - 填入 FUTU_COOKIE 和 ACCOUNT_ID

4️⃣  启动服务
   双击运行: start.bat
   或执行: python main.py

5️⃣  访问文档
   http://localhost:8000/docs

💡 使用示例
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

查询账户:
   curl http://localhost:8000/api/account/info

搜索股票:
   curl -X POST http://localhost:8000/api/stock/search \
     -H "Content-Type: application/json" \
     -d '{"keyword":"NVDA","market_type":"US"}'

买入美股:
   curl -X POST "http://localhost:8000/api/trade/buy?\
     stock_code=NVDA&market_type=US&quantity=10&\
     price=202.50&order_type=LIMIT"

买入港股:
   curl -X POST "http://localhost:8000/api/trade/buy?\
     stock_code=00700&market_type=HK&quantity=100&\
     price=380.00&order_type=LIMIT"

买入A股:
   curl -X POST "http://localhost:8000/api/trade/buy?\
     stock_code=600519&market_type=CN&quantity=100&\
     price=1650.00&order_type=LIMIT"

查询持仓:
   curl http://localhost:8000/api/positions

⚠️  重要提示
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🔒 Cookie安全
   - Cookie包含敏感信息，不要泄露
   - 不要提交.env文件到代码仓库
   - Cookie会过期，需要定期更新

📊 仅限模拟
   - 本服务仅用于模拟交易
   - 不涉及真实资金
   - 模拟结果不代表实盘结果

🚦 请求频率
   - 建议控制在每秒10次以内
   - 避免过于频繁的请求
   - 注意API限流
